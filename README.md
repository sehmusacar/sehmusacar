## 📱 Featured Projects

### 🎓 SınavBank - Comprehensive Exam Preparation App
A feature-rich Android application designed to help Turkish students prepare for various national exams.

**Key Features:**
- 📚 **Multiple Exam Types**: YKS, LGS, KPSS, ALES, YDS, and more
- 🎯 **Progress Tracking**: Detailed analytics and performance metrics
- 📊 **Statistics Dashboard**: Visual progress reports and study insights
- 🔥 **Firebase Integration**: Real-time data sync and crash reporting
- 💰 **In-App Purchases**: Premium features and content
- 📱 **Modern UI**: Material Design 3 with smooth animations
- 🎵 **Focus Mode**: Pomodoro timer and study music integration
- 📖 **Study Materials**: Flashcards, formulas, and topic summaries

**Tech Stack:** Kotlin, Android SDK, Firebase, AdMob, Google Play Billing, Material Design 3
